# GrindMind
An AI agent that tracks your LeetCode progress, analyzes your solutions, and helps you grow.

## Setup
1. Install Docker Desktop
2. Install uv: powershell -ExecutionPolicy ByPass -c 'irm https://astral.sh/uv/install.ps1 | iex'
3. Install Lemma CLI: uv tool install lemma-terminal
4. Run Lemma stack: iwr https://raw.githubusercontent.com/lemma-work/lemma-platform/main/install.ps1 | iex
5. Import the pod: lemma pod import ./grindmind
